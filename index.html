Kadolab SMS API - Python Usage Guide

This document provides a clear guide on how to interact with the Kadolab SMS API using Python. It covers user registration and sending SMS messages.

API Base URL: https://sms-api.kadolab.com/api

1. User Registration

This endpoint creates a new user account and returns an API token for authenticating future requests.

POST /register

Parameters (JSON Body)

Parameter Type Description
name String The user's full name.
email String A unique email address for the account.
password String Password for the account. Minimum 8 characters.
password_confirmation String Must match the password field.
sms_gateway_username String The username from your SMS Gateway mobile app.
sms_gateway_password String The password from your SMS Gateway mobile app.
sms_gateway_url String The URL for the gateway. Should be set to https://api.sms-gate.app/3rdparty/v1/messages.

Python Example: Registering a User

Important: You must include the Accept: application/json header to ensure the API returns a JSON response.

import requests
import json

# The URL for the registration endpoint
url = "https://sms-api.kadolab.com/api/register"

# Data to be sent in the request body
# Remember to use a new, unique email for each registration attempt
payload = {
    "name": "Your Name",
    "email": "your_unique_email@example.com",
    "password": "your_secure_password123",
    "password_confirmation": "your_secure_password123",
    "sms_gateway_username": "YOUR_GATEWAY_USERNAME",
    "sms_gateway_password": "YOUR_GATEWAY_PASSWORD",
    "sms_gateway_url": "https://api.sms-gate.app/3rdparty/v1/messages"
}

# Headers to ensure we get a JSON response
headers = {
    'Accept': 'application/json',
    'Content-Type': 'application/json'
}

# Make the POST request
response = requests.post(url, headers=headers, data=json.dumps(payload))

# Print the response
print(f"Status Code: {response.status_code}")
print("Response JSON:")
print(response.json())

2. Send an SMS

This endpoint sends an SMS message to one or more phone numbers. It requires authentication using the Bearer Token obtained during registration.

POST /send-sms

Authentication

You must include an Authorization header with the token from the registration step.

Format: Authorization: Bearer YOUR_API_TOKEN

Parameters (JSON Body)

Parameter Type Description
phoneNumbers Array of Strings A list of phone numbers in international format (e.g., +255620123456).
message String The text message content to be sent.

Python Example: Sending an SMS

Remember to replace YOUR_API_TOKEN with the actual token you received from the registration step.

import requests
import json

# The URL for the send-sms endpoint
url = "https://sms-api.kadolab.com/api/send-sms"

# Your API token from the registration response
api_token = "YOUR_API_TOKEN_HERE" # e.g., "64|1A9FKiM8TJy3lfrqErGHOZaxNB1sJRT6uS7DkSc6c74efdd7"

# Data to be sent in the request body
payload = {
    "phoneNumbers": ["+255695533100"],
    "message": "Hello from the Python API!"
}

# Headers including the Bearer Token for authentication
headers = {
    'Authorization': f'Bearer {api_token}',
    'Accept': 'application/json',
    'Content-Type': 'application/json'
}

# Make the POST request
response = requests.post(url, headers=headers, data=json.dumps(payload))

# Print the response
print(f"Status Code: {response.status_code}")
print("Response JSON:")
print(response.json())

Success Response (200 OK)

A successful request will queue the SMS for sending and return a status message.

{
  "status": "queued",
  "message": "SMS is being sent in the background."
}
